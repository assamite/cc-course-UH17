Computational Creativity, University of Helsinki, fall 2017¶

The documentation is largely in development. Expect constant changes.

This documentation holds tips and sample code relevant to the course. All the code is written with Python 3.5 in mind. Although many of the snippets work on earlier Python versions, we recommend installing Python 3.5 straight away (see Setup Development Environment).
